This folder contains the files needed to replicate training run 4 
Training run 4 is the training run for the 2019 yellow taxis dataset referenced in the main folders README file without weather data. It additionally has some features with the lowest importances removed and was conducted to investigate the effect that removing features of low importance would have on the models prediction accuracy.

The following notebooks are enclosed 

Filtertomanhatten - filters out the non manhatten aspects of the data.
cleaning_2019 - basic cleaning for 2019 dataset 
featureselector1_3 - further cleaning and selection of features 
feature_complete4_3 - further cleaning and encoding the features.
4_3train - the file for the forth training run. Includes test metrics.
