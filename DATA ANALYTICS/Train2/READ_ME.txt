This folder contains the files needed to replicate training run 2
Training run 2 was conducted with the weather data referenced in the main folders README file in order to test the effect weather data has on the models prediction accuracy.

This is the basic training run using the 2019 dataset without the weather data.

The following notebooks are enclosed 

Filtertomanhatten - filters out the non manhatten aspects of the data.
new_york_weather_final - final cleaned weather data for 2019 
cleaning_2019 - basic cleaning for 2019 dataset 
featureselector1_3 - further cleaning and selection of features 
addingweather2_2 - adds weather to the dataset 
feature_complete2_2 -  encoding the features.
2_2train - the file for the second training run. Includes test metrics.
