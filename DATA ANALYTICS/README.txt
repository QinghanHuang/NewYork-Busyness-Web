-This folder contains a number of sub-folders. Each sub-folder represents
a training run and all the code needed for data cleaning and training the 
machine learning model. 

-NB Each folder also has its own unqiue README file in order to further contexualise the Data Analytics. 

-The CSV files and PKl files have been linked in the 
README files in each subfolder and can be accessed from the google drive 
they are stored in if necessary.



-The main datasets used include 

the 2019 NYC Yellow taxi trips data, available at:
https://data.cityofnewyork.us/Transportation/2019-Yellow-Taxi-Trip-Data/2upf-qytp

the 2018 NYC Yellow taxi trips data, available at:
https://data.cityofnewyork.us/Transportation/2018-Yellow-Taxi-Trip-Data/t29m-gskq

Openweather historical weather data (enclosed in training run 2 where it is used) 
NYC taxi Zones data (enclosed in this folder)



-Also enclosed in this folder are data quality reports for the relevant datasets 
used for the project and the notebooks used to generate the data quality reports.
