This folder contains the files needed to replicate training run 5

This is the basic training run using the 2019 dataset without the weather data but additionally with different hyperparameter choice. This training run was conducted to investigate how a different hyperparameter decision might affect model accuracy.

The following notebooks are enclosed 

Filtertomanhatten - filters out the non manhatten aspects of the data.
cleaning_2019 - basic cleaning for 2019 dataset 
featureselector1_3 - further cleaning and selection of features 
feature_complete1_3 - further cleaning and encoding the features.
5_3train - the file for the fifth training run. Includes test metrics.
